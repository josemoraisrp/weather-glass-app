# 🌤️ Weather App

Aplicação de previsão do tempo desenvolvida com HTML, CSS e JavaScript, consumindo dados em tempo real através da API OpenWeather.

## 🚀 Funcionalidades

- Busca de cidades em tempo real
- Consumo de API REST
- Exibição de:
  - Temperatura
  - Umidade
  - Velocidade do vento
  - Condição climática
- Interface moderna com Glassmorphism
- Vídeo de fundo dinâmico
- Layout responsivo

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- OpenWeather API

## 🎨 Design

O projeto utiliza:
- Glassmorphism
- Fundo em vídeo
- Interface moderna
- Efeitos visuais suaves

## 🌎 API Utilizada

OpenWeather API:
https://openweathermap.org/api

## 📱 Responsividade

Aplicação adaptada para:
- Desktop
- Tablet
- Mobile
